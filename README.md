# Voio Releases

Public release artifacts for [Voio](https://voio.app) — the AI-powered meeting assistant.

This repository contains compiled installers and auto-updater manifests only.  
Source code is maintained in a private repository.

## Downloads

Visit [voio.app](https://voio.app) to download the latest version.

## Auto-updater

Voio uses the Tauri updater. The latest release manifest is served at:

```
https://releases.voio.app/latest.json
```
